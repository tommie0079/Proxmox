Videresende kamera port til vm i proxmox
1.	Koble på kamera og identifiser den på proxmox host. Gå til proxmox -> Datacenter -> pve og trykk her på Shell.
Lsusb
No skal me vidaresende usb porten av kameraet til Zoneminder vm. Endre vm id og hardware id til for eksempel:
qm set 105 -usb0 host=046d:0843


	Viss du vil kan du videre koble fleire kamera slik:
qm set 103 -usb0 host=046d:0843
qm set 103 -usb1 host=17ef:482f
Sjekk om config er rett og ta ein omstart av vm-en om kameraet ikkje kjem i devices grep usb /etc/pve/qemu-server/103.conf
Viss ikkje alle kamera kjem opp, sjekk drivere!
