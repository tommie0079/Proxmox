# Proxmox

Proxmox Virtual Environment, usually called Proxmox VE, is an open-source virtualization platform used to run and manage virtual machines and containers.

It combines KVM for virtual machines, LXC for containers, web-based management, storage tools, backup options, and clustering features in one platform.

This repository contains simple Proxmox-related notes and guides, including setup steps and configuration examples.
